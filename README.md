# Microsoft Teams Tab: WebEx Outlook (O365) Integration

## Setup

```
Create App at apps.dev.microsoft.com
Configure properties.ts files in SRC and server>services>properties.ts
$ npm install
```

## Running

```
$ npm start gulpWatch
$ npm run build:dev
```

## Build

```
$ npm run build:prod
```
